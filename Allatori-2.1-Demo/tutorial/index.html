Allatori Obfuscator Tutorial

Here we will learn how to obfuscate Java applications using Allatori Obfuscator. We will obfuscate a Mouse Gestures library. The library was developed by Smardec and is an open-source software. The original version of the Mouse Gestures library used in this tutorial (MouseGestures-1.2.zip) can be found in the tutorial/MouseGestures directory of the Allatori distribution. This tutorial doesn't cover all Allatori features and doesn't use all it's potential. It is only the first little steps to the world of obfuscation and protected software.
